# AI-Solutions Mermaid Diagrams

These `.mmd` files are based on the AI-Solutions product:

- `use-case-diagram.mmd`
- `class-diagram.mmd`
- `sequence-diagram.mmd`
- `architecture-diagram.mmd`
- `database-design.mmd`
- `erd-diagram.mmd`
- `data-flow-diagram.mmd`

Each file contains one diagram so it is easier to paste into draw.io, resize, and capture for your report.

## Use in draw.io

Open draw.io, then go to:

`Arrange > Insert > Advanced > Mermaid`

Paste the content of one `.mmd` file and click `Insert`.

## Use in Figma

Paste the Mermaid code into https://mermaid.live, export as SVG, then drag the SVG into Figma.
