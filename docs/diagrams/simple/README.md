The class diagram and live chat sequence diagram can be added if your teacher asks for extra UML diagrams.

## How to Use in draw.io

1. Open draw.io.
2. Go to `Arrange > Insert > Advanced > Mermaid`.
3. Copy one `.mmd` file.
4. Paste and click `Insert`.
5. Export as PNG and add it to the report.

## How to Use in Figma

1. Open https://mermaid.live.
2. Paste one `.mmd` file.
3. Export as SVG.
4. Drag the SVG into Figma.
